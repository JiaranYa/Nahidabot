## 指令集
  **绑定 [uid]：绑定原神账号**
  
  一个QQ号原则上只绑定一个uid，换绑只需要重新输入这个命令就可以，换绑后，原uid的记录会被保留

  **nhd [角色名]：查询角色面板**
  
  **set [角色名]：更改角色设置**

  [序号1]-[更改值1] 空格 [序号2]-[更改值2]...

  例：W0-200 S1-0
